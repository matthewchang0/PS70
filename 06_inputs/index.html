Electronic Inputs

Reading sensors and turning physical changes into data.

Assignment

This week was about electronic inputs: building a capacitive sensor and using or calibrating another sensor so the microcontroller could read something from the environment.

Capacitive Sensor

The capacitive sensor documentation should include the circuit, the material used as the sensing surface, sample serial readings, and the threshold or mapping used to turn readings into behavior.

Second Sensor

This section is ready for the second sensor calibration: what the sensor measures, how it was wired, how the values changed, and how the data was interpreted.

Reflection

The strongest version of this writeup will include raw readings, calibrated readings, and a short explanation of how noisy or reliable the sensor felt in practice.
