Hand Tools and Fabrication

Building a kinetic sculpture and learning how the physical parts behave.

Assignment

The Week 3 assignment was to make a kinetic sculpture using hand tools and fabrication techniques. The project was about motion, mechanism, and the small tolerances that make a moving object feel smooth instead of stuck.

Process Notes

When documenting this week, the most useful details will be where the motion came from, how the parts were joined, where friction showed up, and what changed between the first test and the final sculpture.
